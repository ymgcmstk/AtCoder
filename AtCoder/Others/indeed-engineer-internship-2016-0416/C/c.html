C - マス目と駒

Time limit時間制限 : 2sec / Stack limitスタック制限 : 256MB / Memory limitメモリ制限 : 256MB

問題文

6 × 6 のマス目があります。 いくつかのマスに駒を置いて、次の条件を満たすようにしたいです。
  • 各マスには 0 個または 1 個の駒が置かれている。
  • 各列にはそれぞれちょうど 3 個ずつの駒が置かれている。
  • 各行にはそれぞれちょうど 3 個ずつの駒が置かれている。
すでにいくつかのマスには駒が置かれている可能性があります。 s_{i, j} が o のとき、上から i 番目、左から j 番目のマスには駒がすでに置かれています。 そうでないとき、s_{i, j} は . であり、このマスに駒は置かれていません。

駒が置かれていないマスのうちいくつかに駒を置いて、条件を満たすようにする方法が何通りあるか求めてください。 ただし、最終的に駒が置かれたマスの集合が異なるとき二つの方法が異なるとみなすことにします。

制約

  • s_{i, j} は o または . である。

入力

入力は以下の形式で標準入力から与えられる。

s_{1, 1} … s_{1, 6}
:
s_{6, 1} … s_{6, 6}

出力

駒が置かれていないマスのうちいくつかに駒を置いて、条件を満たすようにする方法が何通りあるか出力せよ。

入力例1

......
.o....
...o..
...o..
......
......

出力例1

32692

入力例2

oooo..
......
......
......
......
......

出力例2

0
すでに最初の行には四個の駒が置かれているので、条件を満たす方法はありません。
